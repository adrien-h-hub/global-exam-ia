## 🌟 Overview

**GlobalExam AI** is a cutting-edge automation system that combines artificial intelligence, computer vision, and adaptive automation to create an intelligent solution for web application interaction. Built with enterprise-grade security and global accessibility in mind.

### ✨ Key Highlights
- 🔐 **Enterprise Security** - Server-controlled access with global authentication
- 🧠 **AI Intelligence** - Advanced OCR and machine learning capabilities  
- 🎯 **Adaptive Automation** - Dynamic element detection with 85%+ accuracy
- 🌐 **Global Deployment** - Heroku-powered worldwide accessibility
- 🔧 **Auto-Installation** - One-click setup for seamless deployment

---

## 🚀 Quick Start

### One-Click Installation (Recommended)

```bash
# 1. Clone the repository
git clone https://github.com/adrien-h-hub/global-exam-ia.git
cd global-exam-ia

# 2. Windows: Double-click to install and run
INSTALL_AND_RUN.bat

# Or use Python directly
python auto_setup.py
```

### Manual Installation

```bash
# Install dependencies
pip install -r requirements-client.txt

# Run the application
python run_app.py
```
